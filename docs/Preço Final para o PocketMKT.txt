### **PocketMKT: Qual mensagem se encaixa em cada tipo?**

- **Utility**: Todas as notificações automatizadas de prazos, agendamento, alterações, relatórios, envio de documentos, confirmações, etc.
- **Authentication**: Somente envio de código de autenticação (caso o PocketMKT implemente login/2FA por WhatsApp).
- **Marketing**: Promoção de produtos/serviços, convites para eventos, pesquisas, upsell — normalmente não usado para avisos de processos/serviços.

> **Resumindo:**  
> **Quase tudo do painel do advogado é Utility (R$ 0,065 por mensagem),** só usando Marketing (R$ 0,37) se enviar propaganda/convite.  
> Use Authentication apenas para mensagens de código de acesso.
